# Muestra de Input — system-audit

Tengo un sistema de registro de clientes construido con Node.js + Express + MySQL.
El sistema tiene 3 años y no se ha actualizado. Los usuarios reportan que es lento
cuando hay muchos registros. No tenemos tests. Las contraseñas están guardadas en MD5.
El código está todo en un archivo app.js de 3000 líneas.
