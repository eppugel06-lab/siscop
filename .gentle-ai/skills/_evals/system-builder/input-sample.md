# Muestra de Input — system-builder

Necesito crear una web app para gestión de presupuestos. El sistema debe permitir:
1. Crear presupuestos con partidas (descripción, cantidad, precio)
2. Calcular totales con IVA incluido
3. Enviar presupuestos a clientes por email
4. Que el cliente pueda aprobar o rechazar online
5. Dashboard con métricas (total aprobados, pendientes, rechazados)

Es para una empresa pequeña en México. 3 usuarios iniciales (admin y 2 vendedores).
No tengo preferencia de stack. Quiero algo moderno y fácil de mantener.
